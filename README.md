# Superfans Mobile UI Components

A clean Flutter component workspace using Widgetbook.

## Current Catalog

- Foundation color palette
- Typography playground with mobile device preview
- Type scale reference

## Setup

```sh
flutter pub get
```

## Check

```sh
flutter analyze
```

## Run

```sh
flutter run -d chrome
```

For a browser-server URL:

```sh
flutter run -d web-server --web-hostname=127.0.0.1 --web-port=54213
```

## GitHub Pages

Widgetbook deploys from `.github/workflows/deploy-widgetbook.yml`.

After pushes to `main`, the public catalog is published at:

```text
https://aadhavan-sf.github.io/superfans-mobile-ui-components/
```
